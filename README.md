# 🏴 VPN Configs | Оптимизированный набор VLESS

## 📌 Описание
Оптимизированный набор конфигураций VLESS для обхода блокировок. 
Собрано 60 рабочих серверов в 18 странах мира.

## 📊 Статистика
- **Всего серверов:** 60
- **Стран:** 18 + Anycast
- **Протокол:** VLESS
- **Формат:** Подписка для импорта
